extra例子.html: 演示使用扩展词库的一种情况。
				直接将该文件拖放到“文本转换”页面左边的文本框即可。
extra例子_extra.xml : 配套“extra例子.html”文件的词库。
				测试时，需要在“文本转换”页面中：
					1、勾选“使用扩展词库”
					2、不勾选“拼音显示在字后面”
extra例子_extra_append.xml : 配套“extra例子.html”文件的词库。
				测试时，需要在“文本转换”页面中：
					1、勾选“使用扩展词库”
					2、勾选“拼音显示在字后面”
测试时，还可以考虑不勾选“使用基本词库”来看看效果。
